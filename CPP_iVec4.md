- int **y**
- int **z**
- int **w**

## Constructors ##
- **iVec4**()
- **iVec4**(const int x)
- **iVec4**(const int x, const int y)
- **iVec4**(const int x, const int y, const int z)
- **iVec4**(const int x, const int y, const int z, const int w)
