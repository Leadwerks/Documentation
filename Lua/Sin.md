# Sin

This function returns the sine of the specified value.

## Syntax

- number **Sin**(number angle)

Parameter | Description
---|---|
angle | angle, in degrees

## Returns

Returns the sine.

## Example

```lua
angle = -65.0

s = Sin(angle)
c = Cos(angle)
t = Tan(angle)

Print("Sin: " .. ASin(s))
Print("Cos: " .. ACos(c))
Print("Tan: " .. ATan(t))
```
