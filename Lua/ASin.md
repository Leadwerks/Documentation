# ASin

This function returns the arc sine of the specified value.

## Syntax

- **ASin**(value: number): number

Parameter:
- value: sine value

Returns:
- Returns the arc sine.

## Example

```lua
angle = -65.0

s = Sin(angle)
c = Cos(angle)
t = Tan(angle)

Print("Sin: " .. ASin(s))
Print("Cos: " .. ACos(c))
Print("Tan: " .. ATan(t))
```
