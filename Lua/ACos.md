# ACos

This function returns the arc cosine of the specified value.

## Syntax

- **ACos**(value: number): number

## Parameters

- **value**: The cosine value.

## Returns

Returns the arc cosine.

## Example

```lua
angle = -65.0

s = Sin(angle)
c = Cos(angle)
t = Tan(angle)

print("Sin: " .. ASin(s))
print("Cos: " .. ACos(c))
print("Tan: " .. ATan(t))
```
