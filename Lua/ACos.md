# ACos

This function returns the arc cosine of the specified value.

## Syntax

- number **ACos**(number value)

Parameter | Description
---|---|
value | the cosine value 

## Returns

Returns the arc cosine.

## Example

```lua
angle = -65.0

s = Sin(angle)
c = Cos(angle)
t = Tan(angle)

Print("Sin: " .. ASin(s))
Print("Cos: " .. ACos(c))
Print("Tan: " .. ATan(t))
```
