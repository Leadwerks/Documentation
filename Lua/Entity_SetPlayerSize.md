# Entity:SetPlayerSize

This method sets the size when player physics mode is in use.

## Syntax

- **SetPlayerSize**(number height, number radius, number stepheight)

| Parameter | Description |
|---|---|
| height | player height, default is 1.8 |
| radius | player radius, default is 0.4 |
| stepheight | player step height, default is 0.51 |
