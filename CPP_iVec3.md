# iVec3 #
This class describes a three-dimensional integer vector.

## Members ##
- int **x**, **width**
- int **y**, **height**
- int **z**, **depth**

## Constructors ##
- **iVec3**()
- **iVec3**(const int x)
- **iVec3**(const int x, const int y)
- **iVec3**(const int x, const int y, const int z)
