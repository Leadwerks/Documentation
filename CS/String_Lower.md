# String.ToLower #
This method returns the lowercase version of the string.

- [String](String.md) **ToLower**()

## Returns ##
Returns the string, converted to lowercase.

## Example ##
```csharp
using System;
class Program
{
    static void Main(string[] args)
    {
        String s = "John Smith";
        Console.WriteLine(s.ToLower());
    }
}
```
