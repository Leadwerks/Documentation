# String.ToUpper #
This method returns the uppercase version of the string.

- [String](String.md) **ToUpper**()

## Returns ##
Returns the string, converted to uppercase.

## Example ##

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        string s = "John Smith";
        Console.WriteLine(s.ToUpper());
    }
}
```
