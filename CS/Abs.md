# Abs

This functions returns the absolute value of a number.

## Syntax

- float **Abs**(float value)
- double **Abs**(double value)
- int **Abs**(int value)

## Returns

Returns the absolute value.

## Example

```csharp
using System;

namespace Example
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(Math.Abs(-3));
        }
    }
}
```
