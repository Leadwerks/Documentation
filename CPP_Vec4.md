# Vec4 #
This structure describes a four-dimensional vector.

## Constructors ##
- **Vec4**()
- **Vec4**(const float x)
- **Vec4**(const float x, const float y)
- **Vec4**(const float x, const float y, const float z)
- **Vec4**(const float x, const float y, const float z, const float w)

## Members ##
- float **x**, **r**
- float **y**, **g**
- float **z**, **b**
- float **w**, **a**
